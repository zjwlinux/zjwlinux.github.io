Node.js 入门

Posted on 2020-05-25 Edited on 2020-05-26 In Node.js

Node.js 安装

  1. 从官网下载安装包：https://nodejs.org/en/

  2. 安装时会安装 node.js 和 npm（包管理工具）

    Mac: 
    Node.js v12.16.3 to /usr/local/bin/node
    npm v6.14.4 to /usr/local/bin/npm

  3. 升级 node.js

    查看本机当前Node.js和npm版本

    $ node -v
$ npm -v

    清除node.js的cache

    $ sudo npm cache clean -f

    安装”n”版本管理工具，管理node.js

    $ sudo npm install -g n

    更新node.js版本

    $ sudo n stable

    更新npm版本

    sudo npm install npm@latest -g

npm

命令

查看当前已安装插件

$ npm list

更新全部插件 

$ npm update [--save-dev]

更新特定插件

$ npm update <name> [-g] [--save-dev]

卸载特定插件

$ npm uninstall <name> [-g] [--save-dev]

加快 npm 安装速度

cnpm

安装 cnpm 

$ npm install cnpm -g --registry=https://registry.npm.taobao.org

测试 cnpm 是否安装成功

$ cnpm -v

使用国内镜像插件库

npm 的默认插件库是 https://registry.npmjs.org/

查看当前使用的 npm 插件库

$ npm config get registry

替换使用国内的 npm 插件库

$ npm config set registry https://registry.npm.taobao.org
